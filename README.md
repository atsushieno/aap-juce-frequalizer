It is a port of [ffAudio/Frequalizer](https://github.com/ffAudio/Frequalizer) Free to [Audio Plugins For Android](https://github.com/atsushieno/aap-core), using [aap-juce](https://github.com/atsushieno/aap-juce/).

Frequalizer is one of a few JUCE effect plugin port.

Frequalizer is based on an ancient JUCE 6.x so this port might be abandoned at some stage.
